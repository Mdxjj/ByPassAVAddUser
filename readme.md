# 绕过杀软添加用户 By Golang

ByPassAVAddUser.exe会默认添加一个用户（testnb$/123@pass）

目前在火绒和360上进行测试，可以绕过

ByPassAVAddUserCli.exe支持自定义用户名和密码

用法：

ByPassAVAddUserCli.exe -u admin -p password

```
Usage of ByPassAVAddUserCli.exe:
  -p string
        密码
  -u string
        用户名
```
